## 日记

今天在群里问了一个问题

`task.js`
```js
new Promise(function (resolve) {
  resolve();
}).then(function () {
  console.log(8);
});
process.nextTick(function () {
  console.log(6);
});
```

我本地运行`node task.js`

为啥输入结果是`8 6`,而不是`6 9`,百思不得其解

然后群友分享了一篇文章

恍然大悟，原来我把脚本放在了`src/scripts`目录下，而`src`目录下是ESM模块，而`node`默认是CJS模块，所以执行顺序不一致

> 在ESM运行时，代码其实是在微任务阶段中，必须清空完微任务队列，才会轮到nextTick
